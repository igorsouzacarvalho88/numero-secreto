
# Readme by Jogo número-Secreto

## Sobre o jogo
> É um jogo de adivinhação de números com a capacidade de trabalhar a agilidade mental e segmentar o pensamento no sentido de produzir as capacidades práticas do celebro e gerar de forma direta o aprendizado.
